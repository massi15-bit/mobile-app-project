# User Stories

1. As a user, I want to create an account so that I can use the app.
2. As a user, I want to log in using my email and password.
3. As a user, I want to see the home screen after logging in.
4. As a user, I want to view details of an item.
5. As a user, I want to save items to favorites.
6. As a user, I want my data to be stored locally.
7. As a user, I want the app to fetch data from an external API.
8. As a user, I want to change app settings.
9. As a user, I want to receive notifications from the app.
